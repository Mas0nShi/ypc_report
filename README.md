# yp_report

yp_report是一个自动化辅助工具，其开放了API接口，可以通过调用API接口来实现打卡的提交。

## Usage

1. clone this repo

    ```bash
    git clone https://github.com/Mas0nShi/ypc_report.git
    ```

2. install requirements

    ```bash
    cd ypc_report
    npm install
    ```

3. run

    ```bash
    npm run start
    ```

## API

1. generate api docs.

    ```bash
    npm run apidoc-release
    ```

2. run again.

    ```bash
    npm run start
    ```

3. open `http://localhost:8080/doc/` in your browser.
